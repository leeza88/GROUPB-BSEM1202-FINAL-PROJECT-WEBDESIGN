# SaloneMedConnect

**SaloneMedConnect** is a modern, responsive multi-page healthcare website designed to connect people in Sierra Leone with trusted doctors, hospitals, healthcare services, and medical information.

## Features

- Responsive design for Desktop, Tablet, and Mobile
- Multi-page website structure
- Modern hospital-inspired UI
- Doctors directory
- Hospital listings
- Medical departments
- Appointment booking page
- Health blog
- Emergency contact section
- Newsletter subscription
- Animated hero section
- Font Awesome icons
- SEO-friendly structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- JSON
- Font Awesome

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/SaloneMedConnect.git
```

2. Open the project folder:

```bash
cd SaloneMedConnect
```

3. Open `index.html` in your browser or use VS Code Live Server.
