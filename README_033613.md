# SaloneMedConnect

> Sierra Leone's Premier Digital Healthcare Platform

SaloneMedConnect connects patients across Sierra Leone with verified doctors, hospitals, clinics, and trusted healthcare resources — all through a modern, accessible digital experience.

---

## 🏥 Overview

SaloneMedConnect is a fully responsive, professional healthcare website built to serve Sierra Leone's unique healthcare needs. It allows patients to:

- Book appointments with verified, SLMDC-licensed doctors
- Browse healthcare professionals by specialty, location, and availability
- Access comprehensive medicine information
- Subscribe to health tips and updates
- Contact the platform for support

## 🔧 Setup

This is a static website — no build tools required.

1. Clone or download the repository
2. Add images to the `images/` folder (see above)
3. Open `index.html` in any modern browser

For local development with JSON data loading:
```bash
# Option 1: Python
python -m http.server 8080

# Option 2: Node.js
npx serve .

# Option 3: VS Code
Install "Live Server" extension and click "Go Live"
```
